# KimiSound — Design Brief Completo

## O Produto

**KimiSound** é um software desktop para DJs com IA que:
- Calcula compatibilidade musical em tempo real (Flow Score 0–100)
- Monta sets automaticamente em 3 passos (DJ Flow Wizard)
- Gerencia biblioteca com análise por IA (BPM, Key, Energia, 180+ gêneros)
- Aprende o estilo musical do DJ ao longo do tempo
- Exporta para Pioneer CDJ/XDJ via USB

**Plataforma:** Windows 10/11 (foco principal), macOS, Linux
**Modelo:** SaaS desktop (Free / Pro R$19/mês / Studio R$49/mês)
**Trial:** 14 dias grátis no Pro, sem cartão de crédito

## Problema que Resolve

| Problema | Solução KimiSound |
|----------|-------------------|
| 3–4 horas preparando sets manualmente | Set completo em 8 segundos |
| Compatibilidade no "feeling" | Flow Score baseado em 5 dimensões técnicas |
| Biblioteca desorganizada | IA classifica automaticamente 180+ gêneros |
| Ferramentas fragmentadas | Tudo em 1: análise + player + flow + export |
